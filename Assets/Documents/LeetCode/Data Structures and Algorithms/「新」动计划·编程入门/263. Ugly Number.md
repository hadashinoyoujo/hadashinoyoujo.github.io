# 263. Ugly Number

```c++
class Solution {
public:
    bool isUgly(int n) {
        if(n == 1) return true;	// 不可省略
        if(n <= 0) return false; // 不可省略
        
        while(n % 2 == 0) n /= 2;
        while(n % 3 == 0) n /= 3;
        while(n % 5 == 0) n /= 5;

        if(n == 1) return true;
        else return false;
    }
};
```

注意：

前面不能省略判断 n == 1，n <= 0 的部分，最后的判断不能等同于开头的判断，因为如果缺少开头的判断，会导致 n % 2 == 0，n /= 2 仍然是 0，进入无限循环。
