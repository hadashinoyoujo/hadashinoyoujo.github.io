# 231. Power of Two

```c++
class Solution {
public:
    bool isPowerOfTwo(int n) {
        if(n == 1) return true;
        if(n <= 0) return false;
        
        if(n % 2 == 0) {
            return isPowerOfTwo(n / 2);
        }

        return false;
    }
};
```

需要注意两点：

1. 1是很特殊的数，它既是2的幂，也是无法被2整除的数（余数为1）。
2. 题目给定n的范围是-2^31 <= n <= 2^31，但幂是不可能小于等于零的，所以也要单独处理。
