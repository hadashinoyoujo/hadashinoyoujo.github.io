# 28. Find the Index of the First Occurrence in a String

Beats 100%，但题目考的是 KMP 算法，不是标准库的用法。

```c++
class Solution {
public:
    int strStr(string haystack, string needle) {
        if(haystack.find(needle) != std::string::npos) return (int)haystack.find(needle);
        else return -1;
    }
};
```
